10 вариантов дизайна — одиночное добавление товара

Цель: меню удобнее, компактнее, интуитивнее. Один экран = один концепт. Откройте файл в браузере и сравните. Поля кликабельны (демо).

1. Essentials 2. Chips 3. Steps 4. Split 5. Command 6. Bottom sheet 7. Rail 8. Smart detect 9. Keyboard 10. Minimal
1

Essentials-first (рекомендую как базу)

Сверху только имя + цена + категория. Всё остальное — «Ещё». Минимум шума.

Почему удобно: 80% добавлений — 3 поля. Остальное не мешает.

2

Quick category chips

Категория = крупные чипы (GPU/CPU/RAM…), без длинного select.

Интуитивно для склада: палец/мышь выбирает тип за 1 клик.

Defekt / for parts
3

3 шага (What → Money → Extra)

Прогресс-бар; на шаге 1 только идентификация товара.

Сейчас шаг 2/3 — деньги. Меньше когнитивной нагрузки для новичков.

Товар: Samsung M378A5143EB1-CPB 4GB · RAM
4

Split: фото слева / поля справа

Компактная карточка как в мессенджере объявлений.

+ Фото
или Ctrl+V

1 фото = достаточно для старта

5

Command-line paste

Одна строка: название | цена | категория — парсится сразу.

Максимально быстро с телефона после покупки на KA.

→ HP ProDesk G4 MT | 45 | PC
Распознано
HP ProDesk G4 MT
PC €45,00 Heute
6

Mobile bottom sheet

Нижняя «шторка»: сегменты What / Buy / Media.

⌕

Большие зоны нажатия · один фокус

7

Icon rail (левая навигация секций)

Секции формы как иконки: Info / € / 📷 / Specs — без длинного скролла.

В наличии (? → present)
8

Smart detect banner

Вставил имя → сразу предложили категорию, specs, похожие из склада.

AI / local detect
Components · RAM
8GB DDR4 Qty ×2?

Похоже в складе: 1× Kingston 8GB DDR4 — открыть?

9

Keyboard-first power user

Tab-порядок + хоткеи. Компактная сетка без декора.

Enter сохранить · Ctrl+Shift+D defekt · / фокус в имя

10

Ultra-minimal one column

Только имя → авто-всё остальное → одна кнопка. Идеал «добавить за 5 секунд».
